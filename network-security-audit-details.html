Network Security Audit Report

A detailed audit of network security, identifying vulnerabilities and mitigation strategies.

Tools Used

  • Nmap
  • Wireshark
  • Windows Firewall

Project Overview

This project involved conducting a thorough network security audit, using tools like Nmap and Wireshark to identify vulnerabilities, and applying security measures based on the findings.

Challenges and Solutions

Identifying and understanding the vulnerabilities was challenging, especially balancing security with functionality. Solutions included implementing firewall rules and reviewing network services to enhance security without disrupting normal operations.

Results

The audit revealed several open ports and potential vulnerabilities, which were mitigated by disabling unnecessary services and securing the network. A final report was generated documenting the findings and recommendations.
