Network Traffic Analysis with Wireshark

A project focusing on analyzing network traffic using Wireshark to understand and detect anomalies.

Tools Used

  • Wireshark
  • TCP/IP Protocols
  • Network Traffic Samples

Project Overview

This project involved using Wireshark to capture and analyze network traffic to detect anomalies, understand data flows, and troubleshoot network issues.

Challenges and Solutions

A challenge was learning how to filter and interpret the traffic patterns. I focused on identifying common attack signatures and understanding protocol behaviors to detect anomalies in the network data.

Results

The project successfully demonstrated how to analyze network traffic using Wireshark, improving my ability to detect potential security threats and troubleshoot network issues effectively.
